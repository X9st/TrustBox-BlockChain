
# 区块链版 “朋友圈盲盒”—— 去中心化社交礼物交换平台
核心玩法：用户可在平台发布 “盲盒动态”，包含一段文字 / 图片 + 一个 “惊喜承诺”（如 “送一张手写明信片”“分享一首私藏歌单”），并质押少量 ETH；其他用户消耗积分（通过点赞 / 评论 / 发布盲盒获得）可 “拆盲盒”，拆到后双方完成承诺交换，平台自动记录履约情况，履约率高的用户解锁 “诚信勋章”。
创意亮点：
用区块链记录盲盒发布、拆解、履约全流程，不可篡改，解决 “承诺兑现” 信任问题；
加入 “惊喜值” 算法，根据承诺稀缺性、发布者履约率计算盲盒价值，高价值盲盒需更多积分解锁；
社交裂变：用户可邀请好友助力 “加速拆盒”，助力者获得积分分成，形成传播链条。
技术核心：
盲盒数据存储在 IPFS（文字 / 图片）+ 链上（元数据：发布者地址、承诺描述、质押金额、解锁条件）；
履约验证：双方完成交换后上传凭证（如明信片照片），由社区节点投票确认（简易 DAO 机制），投票结果上链。


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
